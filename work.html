Freshii Product Design · UX · UI

Redesigning Freshii’s ordering experience for clarity and speed

Context

Freshii is a health‑focused restaurant brand where customers build customized meals. The existing digital experience made it hard to quickly understand options and complete an order.

Problem

Users struggled with too many steps, unclear customization, and a checkout flow that felt longer than it needed to be.

Process

I reviewed the existing journey, mapped key tasks, sketched new flows, created mid‑fi wireframes, and iterated on prototypes based on usability feedback.

Responsibilities

  • User flow mapping and task analysis
  • Wireframes and interaction design
  • Visual design and layout
  • Prototype creation and testing

Outcome

The redesigned flow groups decisions more logically, reduces friction at checkout, and helps users see their choices and prices clearly before paying.
