# Khushi Patel | UI/UX Designer Portfolio

This is a **bright, minimalistic, and fully responsive portfolio website** for Khushi Patel, a UI/UX Designer.  
It showcases her projects, skills, experience, education, and includes a contact form and social links.

---

## Features

- **Hero Section**: Name, title, tagline, portrait, and CTA buttons.
- **About Section**: Designer bio, skills, and tools.
- **Portfolio Section**: Grid of projects with thumbnails and project info.
- **Experience Section**: Professional work experience in a clean timeline.
- **Education Section**: Academic background.
- **Contact Section**: Contact form (demo) and LinkedIn link.
- **Responsive Design**: Works on mobile, tablet, and desktop.
- **Smooth Animations**: Fade-in sections and interactive hover effects.
